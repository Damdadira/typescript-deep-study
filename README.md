# TypeScript + React Study

### 📚 학습 내용
#### 1. Typescript 기초 연습
- 기본 타입(string, number, boolean, array)
- 여러 타입(string | undefined, number | null)
- 지정된 값 타입('red' | 'orange' | 'yellow')
- 함수에 타입 지정
- Interface
- Type으로 별칭 만들고, 두 타입 합치기(교집합 타입)
- Generics

#### 2. React 컴포넌트 타입
- Props에 타입 지정(null인 경우도)
- 함수 타입
- 이벤트 핸들러 타입 지정

#### 3. 상태(State) 타입
- useState, useReducer에 타입 지정

#### 4. Context API 사용
- Context에 타입 추가
- 커스텀 훅 추가
